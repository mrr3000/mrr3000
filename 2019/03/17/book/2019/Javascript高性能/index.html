Javascript高性能

2019-03-17 javascript, 书籍

一、加载和执行

  1. 脚本放到底部
  2. 内嵌脚本放到LINK后会先等样式表下载完后才执行，建议不要这样放置。
  3. 减少页面中的外链JS会改善性能
  4. 无阻塞脚本在于，在页面加载完成后才加载JS代码 defer属性
  5. async defer都是采用并行下载，在下载过程中不会产生阻塞。区别在于执行时机，A是加载完成后自动执行 D是需要等页面完成后执行！
  6. 任何带有defer属性的无论外链内联都是在DOM完成后才会被执行。不会阻塞浏览器的其他进程。
  7. 动态脚本加载成为最通用的无阻塞加载解决方案。
